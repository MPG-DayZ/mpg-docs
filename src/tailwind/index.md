# Tailwind

[Tailwind](https://tailwindcss.com) используется в проектах реплатформинга.

Причины выбора TW - попытка систематизировать css код с помощью внешних инструментов на фоне почти полного отсутствия единых требований и системности в процессах разработки в компании.

Основная выгода от TW - упрощение и ускорение разработки и поддержки кода.

Для упрощения начальной конфигурации проекта используется готовый npm-пакет [@webui/tailwind-config-letu](https://gitlab.letoile.tech/webui/packages/tailwind-config-letu)
