# MPG_BossHorde

::: info Discord
[Поддержка и обсуждение мода](https://discord.com/channels/1198959737820168192/1262523199481188422)  
:::

## Описание

**Серверный мод**, который позволяет настроить любого зомби или животного в качестве босса со своими подопечными.

Когда босс реагирует на игрока, вокруг него начинают появляться подопечные.

Таким образом можно неплохо разнообразить геймплей на сервере.

## Основные возможности мода

- Удобная настройка через JSON конфиг
- Возможность отключить мод
- Логи в отдельный файл
- Настройка каждого босса отдельно
- Настройка списка подопечных, их шанса спавна, здоровья и лайвтайма
- Настройка радиуса появления подопечных вокруг босса
- Настройка уровня агрессии, при котором будут появляться подопечные
- Настройка количества подопечных и задержки первого спавна
- Настройка количества подопечных и задержки последующих спавнов
- Настройка времени удаления убитых подопечных
- Настройка времени удаления подопечных при убийстве босса
- Настройка удаления подопечных, если босс потерял интерес к игроку

При первом запуске сервера мод сам создаёт файлы с json конфигом. После чего вы сможете настроить их так, как вам
нужно.
