# Api Клиент

На проектах реплатформинга было принято решение использовать в качестве api клиента, автоматически генерируемый npm-пакет - [@webui/vi-client](https://gitlab.letoile.tech/webui/packages/vi-client)

Причины такого выбора - упрощение работы с API и более простая поддержка кода api-клиента в актуальном состоянии, которое соответствует [openapi-документации](https://gitlab.letoile.tech/openapi/re-api)

Актуальная openapi-дока находится на стенде https://ecom-master-re-api.k8s-stage.letoile.tech
