# Конфиг мода

Конфиг мода находится в файле `MPG_MagnetArmband/Config.json` внутри папки с профилем сервера.

## configVersion

`configVersion` - Версия конфига, служебное поле, не нужно его менять. Оно обновляется само при обновлении мода, если в
этом есть необходимость.

## AllowList

`AllowList` - Список класснеймов, которые будут собираться в инвентарь. Можно указывать родительский
класснейм. Например Pelt_Base - будут собираться все шкуры

Пример:

```json
"AllowList": [
// Будут собираться все шкуры
"Pelt_Base",
// Будут собираться медвежьи стейки
"BearSteakMeat",
// Будут собираться тряпки
"Rag"
]
```

## MaxRadius

`MaxRadius` - Радиус, в метрах. В указанном радиусе будет происходить поиск предметов для сбора.

Пример: `"MaxRadius": 10,`

## MessageTakedLoot

`MessageTakedLoot` - Текст сообщения с количеством собранных предметов и предметов, которые не
собраны, и лежат рядом с игроком.

`%1` будет заменён на цифру, равную количеству собранных предметов,
`%2` - будет заменён на цифру с количеством перемещённых под ноги предметов.

Пример:

```
"MessageTakedLoot": "Собрано предметов: %1. \nНе уместилось в инвентарь: %2",
```

## MessageCoolDown

`MessageCoolDown` - Текст сообщения с оставшимся временем до перезарядки. %1 будет заменён на
цифру, равную количеству секунд до перезарядки

Пример:

```
"MessageCoolDown": "%1 сек. До перезарядки повязки-магнита.",
```

## CoolDown

`CoolDown` - Количество секунд, требуемое для перезарядки повязки

Пример: `"CoolDown": 10.0,`
