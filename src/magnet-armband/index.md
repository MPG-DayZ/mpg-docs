# MPG_MagnetArmband

::: info Discord
[Поддержка и обсуждение мода](https://discord.com/channels/1198959737820168192/1211589996474273802)  
:::

## Описание

**Клиентский мод**, добавляющий в игру повязку, которая при использовании собирает определённый лут в определённом
радиусе с земли в инвентарь.
А то, что не поместилось в инвентарь - будет перемещено игроку под ноги.

Класснейм повязки: `MPG_MagnetArmband_01`

## Установка

Установить мод можно любым, привычным для вас способом, скачав его со Steam.
