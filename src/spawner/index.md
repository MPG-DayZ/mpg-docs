::: info Discord
[Поддержка и обсуждение мода](https://discord.com/channels/1198959737820168192/1209994533534826506)

## Описание

**Серверный мод**, который позволяет очень гибко, достаточно удобно настроить нестандартные точки спавна животных и
зомби, используя json конфиги и позволяет визуально отображать триггеры во время отладки.

## Основные возможности

- Корректная работа с животными и с зомби
- Загрузка нескольких конфигов
- Визуализация триггеров для дебага (отдельным клиентским модом)
- Нотификации при входе
- Нотификации при выходе, спавне
- Зависимость триггера от других триггеров или от одного из других триггеров
- Автоматическое размещение триггера на поверхности
- Триггеры в виде сферы, цилиндра или куба с настраиваемыми размерами и возможностью поворота в горизонтальной плоскости
- Задержка первого срабатывания триггера
- Задержка срабатывания после входа в триггер
- Расписание работы по игровому времени
- Несколько точек спавна для одного триггера
- Гибкое логирование: глобальный уровень ([logLevel](mod-config.md#loglevel)) + детальный лог отдельных точек ([isLogsEnabled](point-config.md#islogsenabled))
- Логи в отдельный файл
- Архив логов
- Дебаг и визуализация отдельных триггеров
- Подписи над заспавненной живностью для отладки (видят только админы, отдельным клиентским модом)
- Минимальное и максимальное количество спавна
- Переспавн во время нахождения в радиусе действия
- Возможность вращать живность при спавне
- Возможность задать здоровье и лайвтайм для живности
- Лимит на количество живой живности для одного триггера
- Настройка времени удаления животных и зомби после смерти
- Настройка шанса спавна для каждого класснейма
- Группы спавна: именованные наборы класснеймов с условиями (время суток, погода, шанс), переиспользуемые в нескольких точках через ссылку `@groupName`; поддержка вложенных групп и одноразового появления (`spawnOnce`)
- Настройка минимальной дистанции спавна от игрока
- Деактивация и активация деактивированных триггеров
- Очистка живности в триггерах при покидании зоны, при работе по расписанию
- Удобное добавление и удаление маппинга для каждого триггера с гибкими настройками
- Спавн лута с настройками лайвтайма, наполненности (магазины, вода, еда), здоровья и состояния пищи
- Настройка времени удаления лута
- Возможность рандомизации некоторых параметров
- Фильтр срабатывания по предмету: спавн только если у игрока есть определённый предмет (в руках, в слоте или в инвентаре)

## Использование

При первом запуске сервера мод сам создаёт файлы с json конфигами. После чего вы сможете настроить их так, как вам
нужно.  
В папке с модом находятся файлы с детальным описанием каждого из параметров.

::: tip Не знаете, с чего начать?
Если вы ставите мод впервые, загляните в [Быстрый старт](quickstart.md) - там по шагам показано, как за пару минут
сделать первую рабочую точку спавна.
:::
