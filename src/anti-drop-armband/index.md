# MPG_AntiDropArmband

::: info Discord
[Поддержка и обсуждение мода](https://discord.com/channels/1198959737820168192/1211591813711007774)  
:::

## Описание

**Клиентский мод**, добавляющий повязку, которая не даёт предмету в руках падать на землю, когда игрок находится без
сознания.

Класснейм повязки: `MPG_AntiDropArmband_01`

## Установка

Установить мод можно любым, привычным для вас способом, скачав его со Steam.

https://steamcommunity.com/sharedfiles/filedetails/?id=3052481803

## Редактирование характеристик повязки
