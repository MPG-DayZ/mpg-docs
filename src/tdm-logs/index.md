# MPG_TDM_Logs

::: info Discord
[Поддержка и обсуждение мода](https://discord.com/channels/1198959737820168192/1249513754186612806)  
:::

## Описание

**Серверный мод**, который позволяет вести удобное логирование действий игроков с предметами.

Логи можно загрузить на [сайт](https://map.mpg-dayz.ru) для более удобного просмотра и анализа.

## Основные возможности мода

- Работает серверным модом, поэтому игроки не узнают о его наличии на сервере.
- Каждая запись лога настолько краткая, насколько это возможно.
- Логи можно закидывать в excel (разделитель - табуляция)
- Логи можно загрузить на сайт map.mpg-dayz.ru для удобного просмотра, фильтрации прямо на карте
- Отправка сообщений в дискорд о случаях, когда предмет сброшен или подобран с очень большого расстояния

## Установка

::: warning Это серверный мод!
Не нужно устанавливать его на клиенте.
:::

Установить мод можно любым, привычным для вас способом, скачав его со Steam.

https://steamcommunity.com/sharedfiles/filedetails/?id=3248961196

## Использование

При первом запуске сервера мод сам создаёт файлы с json конфигом. После чего вы сможете настроить их так, как вам
нужно.
