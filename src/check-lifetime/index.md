# MPG_CheckLifeTime

::: info Discord
[Поддержка и обсуждение мода](https://discord.com/channels/1198959737820168192/1223562768972779611)  
:::

## Описание

**Клиентский мод**, добавляет на сервер книжку, с помощью которой можно проверить лайвтайм почти любого игрового
объекта.

Время отображается в удобном, для восприятия, виде.

Класснейм проверяемого предмета копируется в буфер обмена.

## Установка

Установить мод можно любым, привычным для вас способом, скачав его со Steam.

https://steamcommunity.com/sharedfiles/filedetails/?id=3099012083

## Использование

- Взять книжку в руки (`MPG_AdminBook_01`).
- Навести курсор на предмет, который требуется проверить
- Выполнить действие
